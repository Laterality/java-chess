# java-chess

# 기능 목록 정리

## 체스 1단계
- [x] 체스판 출력하기
- [x] 입력 : start 시작 , end 종료

## 체스 2단계
- [x] 사용자 입력에 따른 말 이동
    - [x] 목적 위치에 말이 있는 경우
    - [x] 같은 편 말이면 이동할 수 없다.
    - [x] 상대 편 말이면 해당 위치로 이동한다.
    - [x] 체스 규칙에 따라 말이 이동 가능한 위치로만 이동한다.
- [x] 이동한 말을 적용해서 체스판 출력

## 체스 3단계
- [ ] king 이 잡히면 게임 종료
- [ ] status를 입력 받으면 남은 말들의 점수를 계산
    - [ ] 같은 줄에 아군 폰이 겹쳐 있으면 0.5점으로 계산
